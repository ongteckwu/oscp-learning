Practice Box Map

Every skill lesson, tied to concrete machines. Each block lists 2–4 boxes from HTB, Proving Grounds, PEN-200 and others — OS-split, with a one-line tell of the technique it drills. Work them attempt-first (Lesson 02): enumerate yourself, spend a struggle budget, take one nudge, then review.

Read before you queue a box
  • TJ Null's list is the source of truth. This map is curated from it (+ 0xdf's OffSec mapping) for teaching; the live sheet is authoritative on what's currently in-scope. Ask me to verify any box.
  • Most HTB boxes here are retired → they need an HTB VIP subscription (the retired-machines list). For OSCP prep that's the standard move.
  • "PEN-200" boxes aren't individually public anymore. OffSec replaced the named lab with six Challenge Labs — scenario (Medtech, Relia, Skylark) and exam-like (OSCP A, B, C). Proving Grounds Practice is the closest public stand-in to exam style.
  • Don't spoil the chain. Pick boxes you haven't seen a write-up for; only watch IppSec/0xdf in review mode after you finish or exhaust yourself. Each HTB box name below links straight to its 0xdf walkthrough — strictly for that after-the-fact review.

Legend. Source: HTB Hack The Box · PG Proving Grounds Practice · PEN-200 OffSec Challenge Labs · THM TryHackMe · VH VulnHub. OS: AD = Windows + Active Directory · Mixed = multi-host network.

15 · Writing the Report

No new box — pick one you've already rooted and write it up to exam standard (every command, every screenshot, proof files). Run it through the /box-notes skill.

16 · Exam Strategy

No new box — this is decision-making, not a new technique. Rehearse the time-boxing and "when to move on" rules on the Lesson 17 mock-exam sets.
