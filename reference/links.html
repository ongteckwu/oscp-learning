Useful Links

Your launch pad — the external sites you'll reach for constantly, in one place. This mirrors the annotated set in RESOURCES.md as a clickable board. New links land here as we find them; tell me to add one and I will.

Practice & box selection

  • Practice Box Map (this course) — per-lesson recommended boxes mapped to each technique, OS-split, with a one-line summary of each. Start here, then cross-check the live sheet below.
  • TJ Null's OSCP prep list — the canonical tiered box list (HTB / PG / VulnLab). Start every study block here.
  • Hack The Box — realistic exam-style boxes; work the TJ Null HTB tier. Dante Pro Lab for pivoting practice.
  • OffSec Proving Grounds — closest in style and grading to the real exam.
  • TryHackMe — Offensive Pentesting — gentler guided ramp before HTB/PG.
  • 0xdf — OffSec exam HTB lists — which HTB boxes map to each OffSec exam.

Methodology & references

  • HackTricks — the default "what do I do against service X / how do I escalate here" wiki. First stop.
  • PayloadsAllTheThings — payloads/techniques for nearly every attack class, strong AD section.
  • GTFOBins — abusable Unix binaries (SUID/sudo/caps). First stop for Linux privesc.
  • LOLBAS — Windows living-off-the-land binaries (the GTFOBins counterpart).
  • PEASS-ng (LinPEAS / WinPEAS) — the standard privesc enumeration scripts.
  • WADComs — interactive Windows/AD tool+command cheat sheet.
  • PentestMonkey — Reverse Shell Cheat Sheet — the classic one-liner reverse shells.
  • revshells.com — interactive reverse-shell generator (pick shell, IP, port, encoding).

Exploitation & post-exploitation tooling

  • Exploit-DB / searchsploit — the public exploit archive; your CVE-triage instinct, operationalised against a live version.
  • Impacket — the Python suite behind most Windows/AD attacks (psexec.py, wmiexec.py, secretsdump.py, GetUserSPNs.py, GetNPUsers.py).
  • NetExec (nxc) — the maintained successor to CrackMapExec; validate/spray creds and run commands across a subnet.
  • evil-winrm — the go-to interactive WinRM shell once you hold Windows creds or an NTLM hash.
  • BloodHound — graphs AD relationships into attack paths to Domain Admin (collect with SharpHound / bloodhound-python).
  • mimikatz — the canonical Windows credential-dumping tool (LSASS, SAM, tickets).
  • PrintSpoofer / GodPotato — abuse SeImpersonatePrivilege to reach SYSTEM on Windows.

Password cracking & wordlists

  • hashcat — the GPU password cracker; see the example-hashes page to pick the right -m mode.
  • John the Ripper — CPU cracker with the handy *2john helpers (ssh2john, keepass2john, zip2john).
  • CrackStation — fast online lookup for already-known hashes.
  • SecLists — the standard wordlist collection (users, passwords, discovery) bundled with Kali.

Pivoting & tunneling

  • ligolo-ng — modern tun-based pivoting; add a route and use your tools natively (no proxychains).
  • chisel — single-binary TCP tunnel over HTTP; reverse SOCKS when you only have a shell.
  • sshuttle — VPN-like routing over SSH for Linux pivots.

Walkthroughs & teaching

  • IppSec — YouTube — deeply explained HTB video walkthroughs; the best way to learn methodology (use in review mode).
  • ippsec.rocks — search IppSec's videos by technique; jump straight to the segment you need.
  • 0xdf — write-ups — best-in-class written HTB/CTF walkthroughs; pair with IppSec.
