Course contents · The path to OSCP+

Course Contents

The whole journey, from working software engineer to exam-ready — grounded in one target: passing the OSCP+. Work the lessons in order; each ends with a hands-on task on a real box. The roadmap below is the full curriculum: all seventeen lessons, end to end, from the attack chain to a timed mock exam.

"When in doubt, enumerate harder."
How this maps to the exam The exam is 100 points, 70 to pass (full rules). Three standalone machines (60 pts) test enumeration → foothold → privesc — Modules 2–3 below. The Active Directory set (40 pts) is Module 4. Pivoting (Module 5) appears across the network. And the graded report is Module 6. Every module earns specific points — that's why we do them in this order.

Progress: all 17 lessons complete — the full path from the attack chain through standalones, Active Directory, pivoting, and exam strategy to a timed mock exam. From here, the work is reps on real boxes.

Module 1 · Foundations

The operator's mindset — the difference between knowing vulns and owning boxes.

  • ✓ L01 · The OSCP Attack Chain — the five-phase map every box flows through.
  • ✓ L02 · How to Practice a Box — the attempt-first loop that builds real skill. Includes your box-notes template.

Module 2 · Enumeration & footholds

Finding the way in — the first 10 points on every standalone, and most of the work.

  • ✓ L03 · Service Enumeration — turn a port list into a prioritised plan.
  • ✓ L04 · Attacking the Web Service — map any app, shortlist foothold candidates.
  • ✓ L05 · Exploiting the Web Foothold — SQLi, file inclusion, upload-to-shell.
  • ✓ L08 · Password Attacks & Credential Reuse — brute force, spraying, hash cracking, and reusing creds everywhere.
  • ✓ L09 · Footholds Beyond the Web — SMB, FTP, and turning a known-vulnerable service version into a shell.

Module 3 · Privilege escalation

From your first shell to full control — the second 10 points on every standalone.

  • ✓ L06 · Linux Privilege Escalation — from www-data to root: sudo, SUID, cron, capabilities, kernel.
  • ✓ L07 · Windows Privilege Escalation — tokens, services, registry, and the misconfigurations Windows loves: low-priv shell to SYSTEM.

Module 4 · Active Directory (40 exam points)

The largest single block of marks — you can't comfortably reach 70 without it.

  • ✓ L10 · AD Fundamentals & Enumeration — domains, users, BloodHound, and the lay of the land.
  • ✓ L11 · AD Authentication Attacks — Kerberoasting, AS-REP roasting, and password spraying.
  • ✓ L12 · Lateral Movement → Domain Admin — pass-the-hash, credential chains, DCSync, owning the DC.

Module 5 · Pivoting & post-exploitation

Reaching networks you can't touch directly — essential for the AD set.

  • ✓ L13 · Pivoting & Tunneling — chisel, ligolo-ng, SSH tunnels, and proxychains to reach hidden subnets.
  • ✓ L14 · Looting & Credential Harvesting — what to grab after access, and why it unlocks the next box.

Module 6 · Exam readiness

Turning skill into a pass — the part most people underinvest in.

  • ✓ L15 · Writing the Report — the graded deliverable; reproducible steps and proof that earn marks.
  • ✓ L16 · Exam Strategy & Time Management — points-per-hour, when to move on, the one-tool rule, and sleep.
  • ✓ L17 · The Mock Exam — assemble and run the real thing end to end, then score yourself honestly.

How to use this course Read a lesson, then immediately do its practice task on a real box — each lesson now names specific picks, all collected in the Practice Box Map (cross-checked against TJ Null's list). The reading is fast; the doing is where the skill forms. Bring your box notes back and ask your teacher to review them.
This course has a teacher The roadmap is a guide, not a cage — if you want to jump to Active Directory, slow down on privesc, or detour into a box you're stuck on, just say so. Ask for the next lesson, a walkthrough, a deeper dive, or a notes review. The workspace remembers where you are between sessions.
